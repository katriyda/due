## What to build

集成系统原生通知功能，实现提醒触发时的 Windows 通知。

核心内容：
- 集成 notify-rust 库
- 实现系统通知的发送（标题、内容）
- 处理通知点击事件
- 支持通知的自定义样式（可选）

## Acceptance criteria

- [ ] 提醒触发时显示 Windows 系统通知
- [ ] 通知显示提醒标题和内容
- [ ] 通知点击后可以打开应用主窗口
- [ ] 通知在用户未操作时自动消失
