## What to build

实现提醒弹窗和用户操作功能。

核心内容：
- 实现窗口弹窗提醒（显示标题、内容、操作按钮）
- 支持关闭操作（关闭弹窗）
- 支持延后操作（5/15/30/60分钟可选）
- 支持标记完成操作
- 弹窗未操作时自动关闭（可配置超时时间）

## Acceptance criteria

- [ ] 提醒触发时显示窗口弹窗
- [ ] 弹窗包含关闭、延后、完成按钮
- [ ] 延后操作支持选择时间（5/15/30/60分钟）
- [ ] 完成操作标记提醒为已完成
- [ ] 弹窗未操作时按配置时间自动关闭
