## What to build

实现重复提醒功能，支持时间区间设置。

核心内容：
- 实现重复提醒的创建（按固定间隔重复）
- 支持日期范围设置（起止日期，可选）
- 支持每日活跃时段设置（如 9:00-18:00，可选）
- 实现重复提醒的下次触发时间计算
- 支持重复次数限制（可选）

## Acceptance criteria

- [ ] 可以创建按固定间隔重复的提醒
- [ ] 可以设置日期范围（开始日期、结束日期）
- [ ] 可以设置每日活跃时段（开始时间、结束时间）
- [ ] 正确计算重复提醒的下次触发时间
- [ ] 支持重复次数限制，达到次数后自动停止
