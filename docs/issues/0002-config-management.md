## What to build

实现应用配置管理模块，支持 TOML 格式的配置文件读写。

核心内容：
- 定义 AppConfig 结构体（通知方式、默认延后时间等配置项）
- 实现配置文件的读取和写入
- 支持默认配置生成（首次运行时自动创建）
- 配置文件存储在应用数据目录

## Acceptance criteria

- [ ] 首次运行时自动生成默认配置文件
- [ ] 可以读取配置文件并解析为 AppConfig
- [ ] 可以修改配置并保存到文件
- [ ] 配置项包含通知方式、默认延后时间等必要设置
