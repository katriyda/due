## What to build

实现时间解析模块和一次性提醒的触发逻辑。

核心内容：
- 解析本地时间格式（如 2026-06-03 15:00）
- 解析相对时间格式（如 "30分钟后"、"2小时后"）
- 实现定时提醒的触发判断（当前时间 >= 提醒时间）
- 使用 chrono 库处理时间计算

## Acceptance criteria

- [ ] 可以解析本地时间字符串为 DateTime
- [ ] 可以解析相对时间字符串并计算目标时间
- [ ] 能够判断定时提醒是否应该触发
- [ ] 支持中文时间格式（如 "下午3点"）
